# Open Questions

The following questions need human clarification before proceeding:

1. **What specific EMR/EHR system is currently used by the hospital?**
2. **Are there any existing APIs or integration protocols available for EMR/EHR systems?**
3. **How many departments will be supported initially, and what are their unique scheduling requirements?**
4. **Is there a preferred vendor or cloud infrastructure provider (e.g., AWS, Azure)?**
5. **What is the expected timeline for full deployment and rollout across all departments?**
6. **Are there any legacy systems that must be phased out during migration?**
7. **Will the system support offline mode for critical functions in case of network failure?**
8. **Should patient consent forms be digitally signed and stored within the system?**
